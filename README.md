Проект разрабатывается для портфолио <br>
Макет предоставлен HTMLAcademy из общей email-рассылки

Используемые технологии:
<b>gulp, scss, html, js</b>

Проект можно открыть в браузере из <b>build/index.html</b>

Проект можно скомпилировать используя следующие команды:<br>
<b>npm i</b> - установка зависимостей<br>
<b>npm run build</b> - сборка проекта<br>
<b>npm run start</b> - запуск проекта<br>
