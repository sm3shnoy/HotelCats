Проект разрабатывается для портфолио
Макет предоставлен HTMLAcademy из общей email-рассылки

Используемые технологии:
gulp, scss, html, js

Проект можно открыть в браузере из build/index.html

Проект можно скомпилировать используя следующие команды:
npm i - установка зависимостей
npm run build - сборка проекта
npm run start - запуск проекта
